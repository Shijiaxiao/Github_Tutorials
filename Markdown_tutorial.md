# How to Write a Markdown

## Title
``` markdown
# Title1
## Title2
### Title3
```

## Text
* Italics: Wrap the text with "*" or "_".
* Bold: Wrap the text with "**" or "__".
  
## Links
``` markdown
[TEXT](Link URL)
```

## List
``` markdown
1. Project1
2. Project2
* Tip1
* Tip2
```

## Reference
Create with ">".
``` markdown
> This is a reference.
```

## Code
Create with "```", you can specify program language.

## Images
``` markdown
![TEXT](Image URL)
```

## Horizontal line
Create with "---", "***" or "___".
