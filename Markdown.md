# How to Write a Markdown

## Title
``` markdown
# Title1
## Title2
### Title3
```

## Text
* 斜体：使用*或_将文本包裹起来，例如 *斜体* 或 _斜体_。
* 粗体：使用**或__将文本包裹起来，例如 **粗体** 或 __粗体__

## Links
``` markdown
[链接文本](链接URL)
```

## List
``` markdown
1. 项目1
2. 项目2
* 项目1
* 项目2
```

## Reference
``` markdown
> This is a reference.
```

## Code
Create with "``` cpp ```", you can specify program language

## Images
``` markdown
![替代文本](图片URL)
```

## Horizontal line
Create with "---"
